## 2024-04-07 - In-place Shuffle in C# using CollectionsMarshal
**Learning:** `Random.Shared.Shuffle(CollectionsMarshal.AsSpan(list))` provides an $O(N)$ in-place shuffle without any additional memory allocations, significantly faster than `list.OrderBy(_ => random.Next()).ToList()`. However, the in-place nature means the caller's list is modified, which requires verification that the original order is no longer needed. Since this list was immediately created before shuffling, it's completely safe.
**Action:** Always prefer `Random.Shared.Shuffle` on a span or array over `OrderBy` with a random key for shuffling collections in modern C#, and be mindful of the in-place mutation. Also make sure to clean up all temporary test files and avoid committing auto-generated test results before submitting.
